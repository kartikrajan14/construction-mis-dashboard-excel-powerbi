# construction-mis-dashboard-excel-powerbi
**Simulated ₹14.25 Cr Metro Viaduct Project**  
Built by Kartikay Rajan | VNIT Nagpur (2025) | GATE 2024 (AIR 8123) 

**Tools:** Excel (Power Query) + Power BI

### Project Objective
Developed a complete end-to-end MIS dashboard for construction project monitoring, replicating real DMRC-style planning workflows used by Planning Engineers and Project Controls teams.

### Key Features Delivered
- Interactive S-Curve showing Planned vs Actual Cumulative Cost
- KPI cards (Overall Progress %, Total Planned Cost, Total Actual Cost, Schedule Variance)
- Material consumption tracking (Planned vs Actual for Cement, Steel, RMC, Aggregates, Precast Segments)
- Activity status breakdown (Completed / In Progress / Not Started)
- Automated cost & schedule variance analysis

### Business Impact
- Reduces manual reporting time by ~65-70%
- Enables faster decision-making on cost overruns and delays
- Single source of truth for stakeholders (exactly what Planning/MIS teams need)
